üöÄ Quick Setup (5 minutes)

  1. Create Firebase Project

    Go to Firebase Console and create a new project. Name it something like "my-notes-app".

  2. Enable Authentication

    In Firebase Console, go to Build ‚Üí Authentication. Click "Get started", then enable Email/Password and Google sign-in methods.

  3. Enable Firestore Database

    Go to Build ‚Üí Firestore Database. Click "Create database" and start in production mode. Choose a location close to you.

  4. Enable Firebase Storage

    Go to Build ‚Üí Storage. Click "Get started" and use production mode. Use the same location as Firestore.

  5. Get Your Configuration

    In Firebase Console, click the Web icon (</>) to add a web app. Register your app and copy the configuration object.

  6. Update Config File

    Open scripts/firebase-config.js in a text editor. Replace the placeholder values with your actual Firebase configuration from step 5.

  7. Set Security Rules

    Copy the security rules from FIREBASE_SETUP.md and paste them into Firestore Rules and Storage Rules in Firebase Console. Click "Publish".

  8. Start Using!

    Open login.html in your browser, create an account, and start taking notes!

‚ö†Ô∏è Important

You must complete the Firebase setup before the app will work. Without proper configuration, you won't be able to login or save notes.
